## 组件选型

| 需求         | 组件                                      | 理由                                 |
| ------------ | ----------------------------------------- | ------------------------------------ |
| 页面标签切换 | `KuiTabs` + `KuiTabPane`                  | 符合项目 Tabs 复合页面惯例           |
| 只读查询表格 | `KuiGeneralView` + `TableView` + `Column` | dict 自动翻译                        |
| 查询表单     | `QueryForm` + `Item`                      | 统一风格、内置 trigger/before-submit |
| 日期选择     | `Item component="date-picker"`            | KJDP 标准日期选择                    |
| 机构选择     | `Item component="org-select"`             | KJDP 机构树选择器                    |
| 分页         | `TablePagination`                         | KuiGeneralView 标配                  |
