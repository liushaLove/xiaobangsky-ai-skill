# 技术栈

## 核心技术

| 类型 | 当前技术 | 说明 |
| --- | --- | --- |
| 模块加载 | `loader.js`、AMD/CMD `define` | 页面主模块由 HTML 的 `data-main` 指定 |
| 框架 | KJDP4 / KUI 4.0.3 | `loader-frame-config.js` 中声明 |
| DOM | jQuery | 组件初始化、事件绑定、AJAX 底层能力 |
| 视图 | Backbone | 业务页面常用 `Backbone.Model`、`Backbone.View` |
| 工具 | `_`、`utils`、`consts`、`cache` | 框架全局工具和缓存 |
| 请求 | `ajax` | `frame/core/ajax.js` 封装 KJDP 后端协议 |
| 组件 | KUI jQuery 插件 | `combobox`、`combotree`、`datagrid`、`form` 等 |
| 模板 | artTemplate / tmod | `apps/tpls/**/*.html` 编译到 `apps/tpls/src/template.js` |
| 样式 | CSS / LESS / SCSS | 页面 CSS、KUI 主题、Bootstrap LESS、frame themes |
| 构建 | Grunt | copy、cssmin、transport、concat、uglify、htmlmin、tmod |

## 不属于当前项目的技术

- Vue 3 / Composition API
- Vue Router
- Pinia
- Vite
- `@vitejs/plugin-vue`
- `@szkingdom.kjdp/core` npm 包导入
- `src/main.js`、`src/App.vue`、`src/portal/**`
