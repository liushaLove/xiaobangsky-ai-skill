# 服务接口

## 请求入口

```javascript
ajax.post({
    req: {
        service: "P9999999",
        bex_codes: "YGT_A1100807",
        CUST_CODE: custCode
    }
}).then(function (data, head, answers) {});
```

## 参数约定

| 字段 | 说明 |
| --- | --- |
| `service` | 后端服务号，必填 |
| `bex_codes` | 业务扩展服务码，常见于 `P9999999` 聚合服务 |
| `noProcess` | 跳过默认业务错误处理 |
| `showErrorPrompt` | 是否显示默认错误弹窗 |
| `extMsgCode` | 扩展成功码 |

## 返回数据

```javascript
var rows = data && data[0] || [];
var row = data && data[0] && data[0][0] || {};
```

## 公共 service

复用优先级：当前页面附近 service、`apps/opp/common/service/*`、`basic/service/*`、页面内直接 `ajax.post`。
