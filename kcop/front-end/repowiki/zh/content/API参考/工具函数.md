# 工具函数

## utils

常见用途：

- `utils.getUrlParam()`：读取 URL 参数。
- `utils.buildUrl(url, params)`：拼接页面跳转参数。
- `utils.toJSON()` / `utils.parseJSON()`：JSON 处理。
- `utils.encrypt()` / `utils.decrypt()`：兼容加解密场景。
- `utils.getPageBsno()`、`utils.getPageBusiCode()`、`utils.getPageMenuId()`：请求头扩展信息。

## consts 与 cache

`consts` 保存登录、请求、菜单、系统常量。`cache` 用于模块和业务数据缓存。基础缓存初始化主要在 `basic/ready.js` 和 `basic/modules/basic-cache`。
