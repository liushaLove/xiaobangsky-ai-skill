# 配置系统

当前配置系统由 loader 配置、框架配置、业务初始化和页面组件配置共同构成。

| 文件 | 说明 |
| --- | --- |
| `loader-config.js` | 项目名、版本、ready 模块、顶层页面白名单、业务 ready |
| `loader-frame-config.js` | KUI 版本、框架加载列表、主题、语言、兼容模式 |
| `basic/ready.js` | 加载公共模块、service、缓存、KUI 默认行为覆盖 |
| `frame/core/ajax.js` | 请求协议、请求头、成功码、会话超时、错误处理 |
| 页面 HTML | `kui-options` 配置组件请求和校验 |

`src/config/index.js`、`src/config/http.js`、`src/config/webapp.js`、`vite.config.js` 不是当前项目配置入口。
