# 故障排除

## 页面空白

检查 HTML 中 `loader.js` 相对路径、`data-main` 是否存在、JS 是否合法 `define(function(require, exports, module){ ... })`、`require` 路径是否 404、控制台是否有 KUI/loader 报错。

## 组件不生效

检查 HTML class 是否为 `kui-xxx`、`kui-options` 字符串是否语法正确、是否在组件初始化前调用了组件 API、id/name 是否被改动但 JS 未同步。

## 请求失败

检查 `req.service`、`bex_codes`、URL 参数、菜单参数、客户机构参数、`head.MSG_CODE`、`head.MSG_TEXT`、traceId，以及是否会话超时或 token 失效。

## 模板不更新

如果改了 `apps/tpls/**/*.html`，需要重新生成 `apps/tpls/src/template.js`。

## 构建失败

检查 Node/Grunt 版本、`node_modules` 是否完整、Grunt 插件是否缺失、是否有语法不兼容的新 JS。
