# 性能优化

`Gruntfile.js` 已包含 `cssmin`、`transport`、`concat`、`uglify`、`htmlmin`、`filerev/usemin`、`tmod` 等优化或产物处理任务。

业务优化原则：

- 初始化请求可复用 `ajax.start()` / `ajax.send()` 合并，但不要破坏已有顺序。
- 大表格优先使用 datagrid 既有分页/查询能力。
- 公共缓存优先走 `basic/ready.js` 和 `basic/modules/basic-cache` 已有机制。
- 不要为了单页性能改动 `loader.js` 或 KUI 全局加载链路。
