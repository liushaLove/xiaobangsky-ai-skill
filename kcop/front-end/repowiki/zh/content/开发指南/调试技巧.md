# 调试技巧

## 页面定位

```powershell
rg -n "data-main|页面标题|字段名|服务号" apps basic frame
```

## 请求调试

查看 `ajax.post` 的 `req.service`、`bex_codes`、业务字段、`noProcess`、`extMsgCode`。返回数据优先打印 `head` 和 `answers`，不要只看 `data[0][0]`。

## KUI 组件调试

```javascript
$("#ORG_CODE").combotree("getValue")
$("#baseInfoWrap").form("validate")
```
