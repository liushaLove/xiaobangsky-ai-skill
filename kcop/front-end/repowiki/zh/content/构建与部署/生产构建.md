# 生产构建

生产构建由 Grunt 默认任务和发布脚本完成。

## 默认处理

- 复制静态资源到 `dist/`。
- 压缩 CSS/JS/HTML。
- 对 CMD 模块执行 transport/concat。
- 替换入口 HTML 中 `data-main`、`data-css` 相对路径。
- 复制产物到 `../../kcop-dist/`。

## 发布前检查

- `dist/` 是否生成完整。
- 入口 HTML 的 `data-main` 是否仍指向正确模块。
- `apps/tpls/src/template.js` 是否为最新。
- Excel/DBF 模板是否被复制。
