# 构建插件系统

当前项目的“插件”主要是 Grunt 插件，不是 Vite 插件。

已使用的 Grunt 插件包括 `grunt-contrib-clean`、`grunt-contrib-copy`、`grunt-contrib-cssmin`、`grunt-cmd-transport`、`grunt-cmd-concat`、`grunt-contrib-uglify`、`grunt-contrib-htmlmin`、`grunt-text-replace`、`grunt-usemin`、`grunt-tmod`、`grunt-contrib-less`。

插件版本和 Node 环境通常与历史工程绑定，不要轻易升级。
