# 构建配置

## Gruntfile.js 核心任务

| 任务 | 说明 |
| --- | --- |
| `clean:build` | 清理 `dist` |
| `copy:web` | 复制 apps/basic/frame/lib/loader 等静态资源 |
| `copy:server` | 复制 `WEB-INF`，如果存在 |
| `cssmin` | 压缩业务和 KUI CSS |
| `transport:kjdp` | 处理框架层 CMD 依赖 |
| `transport:index` | 处理入口页依赖 |
| `concat:kjdp` | 合并框架核心文件 |
| `concat:index` | 合并固定入口 JS |
| `uglify` | 压缩 JS |
| `htmlmin` | 压缩 HTML |
| `tmod:apps` | 编译 `apps/tpls` 模板 |

## 固定入口

`indexJsArr` 包括 `apps/opp/pages/src/index.js`、`apps/opp/operator/pages/src/index.js`、`apps/opp/manager/pages/src/index.js`、`apps/opp/review/pages/src/index.js`、`apps/opp/settlement/pages/src/index.js`。
