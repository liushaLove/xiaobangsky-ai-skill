# 开发服务器

当前项目没有 Vite dev server 配置。页面通常依赖后端容器或静态服务器按实际部署路径访问。

## 本地开发关注点

- 路径必须和页面中 `loader.js` 相对路径匹配。
- 某些能力依赖顶层窗口、登录态、后端网关、native 插件或本地服务。
- 直接打开 HTML 可能只能验证静态结构，无法完整验证登录、缓存、服务请求和 iframe 通信。

常见入口：`apps/opp/index.html`、`apps/opp/operator/index.html`、`apps/opp/manager/index.html`、`apps/opp/review/index.html`。
