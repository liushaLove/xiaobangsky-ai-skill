# 业务流程处理

```javascript
flow.init({
    text: "资料录入",
    el: "#infoNodeWrap",
    init: function ($el, busiData, nodeData) {},
    validate: function () {},
    save: function () {}
});
```

数据流：从 URL 或流程上下文取参数，调用 service 或 `ajax.post` 加载数据，`form("load")` 写入表单，用户编辑 KUI 组件，`validate` 校验，`save/getData` 汇总字段返回给流程框架。
