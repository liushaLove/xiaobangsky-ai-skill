# 后端开发速查

## 常用检索

```powershell
rg -n "<服务码或业务码>" -g "*.xml" -g "*.java"
rg -n "FrameDao\.doAtomCall|FrameDao\.doBexCall" -g "*.java"
rg -n "<字段名或表名>" -g "*.java" -g "*.xml" -g "*.json"
```

## 常用构建

```powershell
mvn clean install -DskipTests
mvn -pl fs-aoi-atom -am clean install -DskipTests
mvn -pl fs-aoi-biz-starter -am clean package -DskipTests
mvn -pl fs-aoi-test -DskipTests=false -Dtest=<TestClassName> test
```

## 注意事项

- `fs-aoi-test` 默认 Surefire `skip=true`，测试运行结果必须确认是否真正执行。
- XML 服务注册链路通常是 service -> ATOM/BEX -> Java/MyBatis。
- 保持 Java 8 兼容。
- 改金融、客户、账户、权限、审核、流程状态时，默认高风险处理。
- 不把券商个性化逻辑随意上提到公共模块。
