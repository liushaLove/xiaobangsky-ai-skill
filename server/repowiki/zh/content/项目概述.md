# fs-aoi-biz 项目概述

`fs-aoi-biz` 是 AOI 业务受理 Java 后端多模块工程。根工程使用 Maven 聚合，Spring Boot 2.5.8 启动，Java 8 编译，业务运行在 KJDP/Koca 体系中。

核心入口是 `fs-aoi-biz-starter/src/main/java/com/szkingdom/fs/AoiServiceApplication.java`。业务服务大量通过 `META-INF/service`、`META-INF/atom`、`META-INF/bex`、`META-INF/mybatis` 这些 XML 元数据注册和路由，不应按普通 REST Controller 项目理解。

主要模块：

- `fs-aoi-biz-starter`：启动、容器 profile、打包。
- `fs-aoi-config`：Spring Java 配置。
- `fs-aoi-frame`：框架公共能力和基础业务。
- `fs-aoi-atom`：主业务 ATOM。
- `fs-aoi-workflow`：流程与审核任务。
- `fs-aoi-access`：准入、检查、白名单。
- `fs-aoi-adapter`：外部、报表、KONE 等适配。
- `fs-aoi-subsys`：子系统模块。
- `fs-aoi-qs`：券商个性化模块。
- `fs-aoi-test`：测试与 mock 支撑。
